# CHANGELOG

本文件记录 `J4Ledger` 的重要变更。

## 2026-05-24

### Added

- 新增导入微信账单功能：支持从微信导出的 .xlsx 账单文件导入消费和收入记录。
- 新增微信账单解析器，自动跳过说明行，提取交易记录。
- 新增微信账单预览 Dialog，支持按交易状态（支付成功、已退款等）多选筛选，默认全选。
- 新增 AI 分类功能：将去重后的"交易类型+交易对方+商品"交给 Agent 判断消费分类。
- 导入的微信账单支付方式统一标记为"微信"。
- 新增 Apache POI 依赖用于 Excel 文件解析。

### Changed

- 合并记账页面消费/收入输入框为单一输入框，由 Agent 自动判断类型。
- 支持一次输入解析多笔账单，例如"吃早饭5块，奶茶13块，工资到账3000"。
- "Dashboard" 改名为 "统计"。
- 移除全局 Header 显示，仅在账单页显示净额和收支汇总。
- 记账页专注于记账功能，不再显示头部概览。
- 底部导航栏贴边显示，移除多余 elevation。

## 2026-05-23

### Added

- 使用 Android CLI `empty-activity` 模板初始化项目。
- 初始化项目协作文档 `AGENTS.md`。
- 初始化项目说明文档 `README.md`。
- 初始化技术说明文档 `技术文档.md`。
- 初始化关键流程文档 `关键链路.md`。
- 新增文本自然语言记账主界面。
- 新增 OpenAI-compatible Agent 解析，支持应用内配置 `base_url`、`api_key`、`model_name`。
- 新增按日期范围查看账单与合计。
- 新增 Agent 本地解析单元测试。
- 新增 SQLite 配置与账单持久化。
- 新增界面标题自定义配置。
- 新增账单长按编辑和删除能力。
- 新增 SQLite 加密 ZIP 数据导出，可选择账单、设置或全选，账单导出支持日期范围。
- 新增 SQLite 加密 ZIP 覆盖导入，导入前提示覆盖风险。
- 新增账单日期区间选择控件，支持今天、本周、本月、本季度、年度快捷选择。
- 新增底部导航栏替代顶部标签，并加入页面切换动画。
- 新增加密 ZIP 备份格式，备份内容使用 AES-GCM 对称加密，导入时自动解压解密。
- 新增显示模式设置，支持跟随系统、亮色和暗色。
- 新增收入记账入口，支持通过 Agent 解析收入类型、收款渠道、金额和日期时间。
- 新增导入配置保护：通过备份导入的 `base_url`、`api_key`、`model_name` 会遮罩并禁用输入。
- 新增 Agent 配置“重置”按钮，可清空界面标题和 Agent 配置并解除导入锁定。
- 新增 `Dashboard` 底部导航页，复用账单日期范围展示每日收支趋势、消费分类占比、收入分类占比、渠道收支对比和收入支出占比。
- 新增 Vico 3.1.0 Compose 图表依赖和 Dashboard 聚合单元测试。
- 新增 Compose Material Icons Extended 依赖，用于底部导航、卡片、按钮和空状态图标。
- 新增 Dashboard KPI 卡片，展示当前筛选范围内的收入、支出、净额和账单数。

### Changed

- 将项目目录、Gradle 项目名、应用显示名和模板主题命名从 `AndroidProject` 调整为 `J4Ledger`。
- 修正 `AGENTS.md` 中过度绑定 Android CLI 和旧项目名的表达，使其更适合后续通过 Codex 持续协作。
- 将最低支持版本调整为 Android 12（`minSdk 31`）。
- 移除语音输入入口和麦克风权限，仅保留文本输入。
- 移除所有本地规则兜底解析；未配置或 AI 服务不可用时仅展示友好提示。
- 数据备份导出文件从 JSON 改为 ZIP。
- 账单数据新增收支类型字段。
- 账单页金额按收支显示正负前缀，收入金额使用绿色强调，顶部合计改为当前筛选净额。
- 移除第一版 SharedPreferences 数据兼容和启动转换逻辑。
- 重做主界面视觉层级：顶部概览卡、底部图标导航、统一卡片留白、按钮按压反馈、账单卡片按压反馈、设置页分组卡片和图表容器。
- 将收入强调色从深黄色调整为绿色，支出继续使用错误色强调。
- 更新主题配色和字体层级，增强浅色/深色模式下的对比度与可读性。
- 更新 README、技术文档和关键链路，记录文本记账、Agent 配置、SQLite、构建与设备运行方式。

### Added

- 新增 `.gitignore` 规则忽略 `*.jks` 和 `*.keystore` 签名文件。

### Changed

- 更新关键链路文档，新增 GitHub Actions 发布链路说明。
